Hackathon Package - GenAI Agents for ETL (World Bank WDI sample)

Contents:
- source_schema.json : JSON schema describing source tables (CSV files)
- source_*.csv : Source CSV files (countries, indicators_meta, gdp, population, life_expectancy, co2_emissions, primary_enrollment, poverty_headcount)
- source_table_descriptions.txt : human-friendly descriptions of source tables
- target_schema.json : JSON describing target BigQuery tables and DDL file names
- target_*.sql : BigQuery DDL scripts for target tables
- target_table_descriptions.txt : human-friendly descriptions of target tables
